# EduTech Backend API

This is a Spring Boot-based backend application for an EduTech platform. The project provides RESTful APIs for managing courses, users, and educational content. It is fully deployed and documented via Swagger.

## üöÄ Live Demo
- **Deployment URL:** [https://edtech-api-20if.onrender.com](https://edtech-api-20if.onrender.com)
- **Interactive API Documentation (Swagger):** [Explore API Endpoints](https://edtech-api-20if.onrender.com/swagger-ui/index.html)

## üõ† Features
- **User Management:** Registration and authentication.
- **Course Management:** CRUD operations for courses and lessons.
- **Validation:** Robust input validation using Spring Boot Starter Validation.
- **Persistence:** Database integration with Spring Data JPA.
- **Testing:** Unit and integration tests for core logic.

## üíª Tech Stack
- **Framework:** Spring Boot 3.x
- **Language:** Java 17+
- **Database:** PostgreSQL/MySQL (as configured in Render)
- **Documentation:** SpringDoc OpenAPI (Swagger UI)
- **Deployment:** Render
